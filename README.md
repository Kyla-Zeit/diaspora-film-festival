# Diaspora Film Festival Website

A modern, sleek website redesign for the Diaspora Film Festival, showcasing diasporic stories through cinema.

## Features

- **Modern Design**: Clean, minimalist aesthetic with bold typography
- **Responsive**: Fully responsive design that works on all devices
- **Animated**: Smooth scroll animations and hover effects
- **Accessible**: Built with accessibility best practices
- **Fast**: Optimized for performance

## Structure

- `index.html` - Main homepage
- `styles.css` - All styling and animations
- `script.js` - Interactive functionality
- `README.md` - This documentation

## Sections

1. **Hero** - Compelling introduction with festival dates
2. **About** - Mission statement and festival statistics
3. **Featured Films** - Showcase of selected films
4. **Volunteer** - Opportunities to get involved
5. **Footer** - Contact information and links

## Colors

- Primary Black: #0a0a0a
- Primary White: #ffffff
- Accent Gold: #d4af37
- Text Gray: #666
- Light Gray: #f8f8f8

## Installation

1. Download all files
2. Open `index.html` in a web browser
3. No additional setup required

## Customization

- Update content in `index.html`
- Modify colors in CSS variables in `styles.css`
- Add new sections following the existing pattern
